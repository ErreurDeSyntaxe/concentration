Concentration: A Game Of Memory

Input Words

Rules of the Game

  1. Flip two cards per turn
  2. Find identical card pairs

How To Use

  1. Input words in the left field
  2. Copy them if you want truly identical cards
  3. Write equivalent meaning in the right field otherwise
